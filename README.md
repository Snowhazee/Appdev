# 🛒 Online Shopping Web App

Web Application สำหรับค้นหาและสั่งซื้อสินค้าออนไลน์

---

## 🚀 Features
- ค้นหาและดูสินค้า  
- เพิ่ม / ลบสินค้าในตะกร้า  
- สั่งซื้อสินค้า (Checkout)  
- ตรวจสอบสถานะคำสั่งซื้อ  

---

## 👥 User Roles
- **Guest**: ดูสินค้า / ค้นหา  
- **Customer**: สมัคร, Login, สั่งซื้อ, ดู Order  
- **Admin**: จัดการสินค้า + คำสั่งซื้อ  

---

## 🛠️ Setup & Run

### 1️⃣ Clone Project
```bash
git clone https://github.com/Snowhazee/Appdev.git
cd Appdev
```
### 2️⃣ Backend
```bash
cd backend
npm install
npm run dev
```
### 2️⃣ Frontend
```bash
cd frontend
npm install
npm run dev
```
